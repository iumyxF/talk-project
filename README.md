# SSE（Server-Send Events）
Server-Sent Events 服务器推送事件，简称SSE，是一种服务端实时主动向浏览器推送消息的轻量级技术。
类似WebSocket，但SSE是基于HTTP协议的，而WebSocket是基于TCP协议的。
如果需要高并发，还是需要使用WebSocket来实现
